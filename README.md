# PowerBI-Project
**Overview**

This project is an interactive Power BI Dashboard built to analyze e-commerce sales performance. It provides business insights using KPIs, charts, slicers, and interactive visualizations to help understand customer behavior, sales trends, and profitability.

**Project Objectives**
1. Analyze total sales performance
2. Track monthly profit trends
3. Compare sales across different states
4. Analyze customer purchasing behavior
5. Study payment mode preferences
6. Identify profitable product categories

**Key Metrics**

| KPI                 | Value |
| ------------------- | ----- |

| Total Profit        | 37K   |

**Insights**
+ Maharashtra generated the highest sales.
+ COD is the most preferred payment method.
+ Clothing contributes the highest quantity sold.
+ Printers generate the highest profit among all sub-categories.
+ November records one of the highest monthly profits.
